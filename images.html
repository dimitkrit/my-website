ΧΑΡΤΟΓΡΑΦΙΚΕΣ Ή ΨΗΦΙΟΓΡΑΦΙΚΕΣ Bitmap graphics Οργανώνονται σε ορθογώνιο πλέγμα από χρωματιστές ψηφίδες, τα εικονοστοιχεία (pixel, picture elements) ΔΙΑΝΥΣΜΑΤΙΚΑ ΓΡΑΦΙΚΑ vector graphics Συντίθενται από γεωμετρικά σχήματα (σημεία, γραμμές, ορθογώνια, ελλείψεις, πολύγωνα)

Ποιές εικόνες επηρεάζει η ανάλυση της εικόνας, η αυξομείωση των διαστάσεών τους

Επηρεάζονται οι bitmap οι ψηφιογραφικές.

Πώς παράγονται οι ψηφιογραφικές εικόνες

  • Από προγράμματα δημιουργίας και επεξεργασίας εικόνας
  • Με ψηφιοποίηση μέσω σαρωτή
  • Με σύλληψη από την οθόνη με τη βοήθεια λογισμικού
  • Από ψηφιακή φωτογραφική μηχανή
  • Με σύλληψη εικόνας βίντεο ή τηλεόρασης

ΑΝΑΛΥΣΗ/ΒΑΘΟΣ ΧΡΩΜΑΤΟΣ/ΜΕΓΕΘΟΣ

ΜΕΓΕΘΟΣ Image size Αριθμός pixel. Προσδιορίζεται από πλάτος*ύψος ΑΝΑΛΥΣΗ Image resolution  Πυκνότητα Διάταξης pixel [Αριθμός pixel ανά μονάδα μήκους] ΒΑΘΟΣ ΧΡΩΜΑΤΟΣ true color Ο αριθμός χρωμάτων bits που είναι διαθέσιμα για τη δημιουργία εικόνας

ΤΥΠΟΣ ΜΕΓΕΘΟΥΣ ΑΡΧΕΙΟΥ

ΧΩΡΗΤΙΚΟΤΗΤΑ ΓΙΑ ΑΠΟΘΗΚΕΥΣΗ bitmap Μέγεθος αρχείου= Πλήθος εκονοστοιχείων*Βάθος /  8 bit per byte bytes = pixel * bit / 8 bit per byte

ΟΡΟΙ ppi, dpi

ppi: pixels per inch [Ανάλυση εικόνας] dpi: dots per inch [Αναλυση εκτυπωτή (κουκίδες μελανιού)]

ΛΟΓΟΣ ΣΥΜΠΙΕΣΗΣ

Εκφράζει το βαθμό που συμπιέζεται ένα αρχείο Όγκος αρχικών δεδομένων / Όγκο μετά τη συμπίεση π.χ. 20:1 = 2MB σε 0,1 MB
